# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Do not file security issues as public GitHub issues.**

Instead, email **security@testmuai.com** with:

- Description of the vulnerability
- Steps to reproduce
- Impact assessment (if known)
- Your contact information

## Scope

This policy covers:

- The kane-cli binary and its behavior
- The installation scripts and Homebrew formula in this repository
- Authentication and credential handling

Thank you for helping keep kane-cli safe.
