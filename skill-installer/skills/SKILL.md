# kane-cli Skill

This skill provides the necessary instructions for AI agents to leverage `kane-cli` for browser automation and related tasks.

## Capabilities

The `kane-cli` allows agents to:
- Perform automated browser tasks.
- Interact with elements on web pages.
- Execute structured workflows involving web interaction.

## Usage

Agents can invoke `kane-cli` commands directly, provided they have access to the CLI in their path or through specific agent-defined mechanisms. Refer to the official documentation for supported commands and integration details.
